# Axodus Institutional Publications

This repository contains public-facing research and institutional documentation
for Axodus, a prototype-stage conceptual infrastructure model for accountable
human--AI coordination, educational participation, governance, and modular
digital institutions.

The repository is intentionally documentation-only. It does not establish that
Axodus is operational, validated, secure, adopted, or ready for deployment.

## Phase 1 status

Phase 1 prepares an editorial foundation and a draft conceptual architecture
and research-agenda paper. Its output is intended for human review, not public
submission or external distribution.

Current work includes:

- editorial and evidence governance;
- a claims register;
- related-work and bibliography records;
- Markdown and LaTeX sources for the first paper.

Whitepapers, derived articles, outreach, grant applications, and submissions
are deferred.

## Repository map

- `governance/`: citation, disclosure, evidence, intellectual-property, and
  review controls.
- `research/`: claims, terminology, gaps, related work, and bibliography
  records.
- `papers/`: draft manuscripts and publication-readiness reports.
- `whitepaper/`, `articles/`, and `outreach/`: deferred-phase indexes only.

## Evidence policy

Repository text must distinguish among externally supported facts, internal
claims, hypotheses, design proposals, and future work. Internal claims do not
enter a public manuscript without evidence or explicit human approval. See
[`governance/evidence-boundaries.md`](governance/evidence-boundaries.md).

## Publication status

Nothing in this repository should be treated as publication-ready unless its
readiness report explicitly says so. The Phase 1 manuscript contains a
correspondence placeholder and other declared blockers.
