# Research Index

This directory contains the controlled research support materials used to
prepare, check, and constrain public-facing Axodus writing.

## Read in this order

1. [claims-register.md](claims-register.md)
2. [terminology.md](terminology.md)
3. [bibliography-verified.md](bibliography-verified.md)

Read the remaining files only when the task needs them:

- [bibliography-candidates.md](bibliography-candidates.md) for not-yet-verified
  sources
- [related-work.md](related-work.md) for internal literature synthesis
- [gaps.md](gaps.md) for unresolved research gaps

## Purpose by file

- `claims-register.md` — tracked internal and evidence-sensitive claims
- `terminology.md` — approved terminology and prohibited drift
- `bibliography-verified.md` — sources ready for controlled manuscript use
- `bibliography-candidates.md` — candidate sources not yet promoted
- `related-work.md` — internal literature synthesis and framing notes
- `gaps.md` — open questions and research gaps

## Agent use rules

- Use this directory to constrain claims, terminology, and references.
- Do not promote a candidate source into manuscript use without checking
  citation policy and metadata verification.
- Do not treat internal synthesis notes as public evidence by themselves.
- If a scholarly assertion lacks support, use `[REFERENCE NEEDED]` rather than
  filling the gap from memory.

## Current status

These files support current manuscript and institutional work, but they do not
by themselves authorize publication, external review, or external circulation.
