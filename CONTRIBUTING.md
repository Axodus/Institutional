# Contributing

This repository accepts documentation and research-writing contributions.
Phase 1 artifacts are drafts requiring human editorial review.

## Before editing

- Read `AGENTS.md` and all files in `governance/`.
- Identify whether a proposed statement is externally supported, an internal
  claim, a hypothesis, a design proposal, or future work.
- Add or update the claims register when a statement affects the public
  description of Axodus.

## References

Add unverified discoveries to `research/bibliography-candidates.md`. Move a
source to `research/bibliography-verified.md` only after checking its metadata
against a primary publisher, journal, standards body, institutional report, or
canonical preprint record. Add only cited sources to a paper's
`references.bib`.

Do not infer a DOI, issue, page range, author list, or publication year.

## Manuscript changes

Update both Markdown and LaTeX sources when changing a paper's substance or
structure. Explain intentional differences in `publication-readiness.md`.

## Review expectations

Contributions must:

- use non-promotional language;
- preserve prototype-stage disclosure;
- avoid operational or empirical claims without evidence;
- expose no sensitive implementation information;
- retain visible publication blockers until resolved by a human reviewer.

No contribution process described here authorizes external publication.
